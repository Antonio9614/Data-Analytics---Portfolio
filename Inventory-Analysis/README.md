# 📦 Inventory Optimization

## Inventory Management & Stock Optimization

### 📌 Business Context

A company aims to optimize its inventory management to reduce operational costs, prevent stock shortages, and improve product availability through data-driven decision making.

### 🎯 Project Objectives

- Perform ABC inventory classification.
- Analyze inventory turnover.
- Identify critical and slow-moving products.
- Detect stock shortages and overstock situations.
- Develop a basic inventory demand forecast.

### 🛠️ Tools

- Python
- Pandas
- Excel
- Power BI

### 📊 Key Performance Indicators (KPIs)

- Inventory Turnover
- ABC Classification
- Stock Availability
- Slow-Moving Products
- Stock Coverage
- Demand Forecast

### 🚧 Project Status

This project is currently under development. The next stages include data collection, data cleaning, exploratory data analysis (EDA), inventory KPI calculation, dashboard development, and business recommendations.
