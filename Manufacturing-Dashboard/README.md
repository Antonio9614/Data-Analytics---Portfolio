Manufacturing Operations Dashboar

Contexto:
Una planta necesita monitorear el desempeño de sus equipos mediante indicadores de mantenimiento y producción.

Objetivos: 
- Analizar fallas.
- Calcular MTTR.
- Calcular MTBF.
- Medir disponibilidad.
- Visualizar tiempos muertos.

Herramientas:
- Python
- Power BI
- SQL
