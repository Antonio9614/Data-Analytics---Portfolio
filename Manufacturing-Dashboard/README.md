# 🏭 Manufacturing Operations Dashboard

## Maintenance & Production Performance Monitoring

### 📌 Business Context

A manufacturing company needs to monitor the performance of its production equipment through maintenance and operational KPIs to reduce downtime, improve equipment reliability, and support data-driven decision making.

### 🎯 Project Objectives

- Analyze equipment failures and maintenance records.
- Calculate MTTR (Mean Time to Repair).
- Calculate MTBF (Mean Time Between Failures).
- Measure equipment availability.
- Visualize downtime and maintenance trends.
- Build an executive dashboard for operational monitoring.

### 🛠️ Tools

- Python
- SQL
- Power BI
- Pandas
- Excel

### 📊 Key Performance Indicators (KPIs)

- MTTR
- MTBF
- Equipment Availability
- Downtime by Machine
- Failure Frequency
- Preventive vs Corrective Maintenance

### 🚧 Project Status

This project is currently under development. The next stages include data collection, data cleaning, exploratory data analysis (EDA), KPI calculation, dashboard creation, and business recommendations.
